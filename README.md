## 🚀 Project Overview & Purpose
This project is a **Progressive Web Application (PWA)** developed for the **WR1 [W.SIE.IN.8011]** course. The goal is to provide a user-centric tool for monitoring daily water consumption, using modern web technologies to bridge the gap between web and mobile apps.

### Core Features:
- **Offline Capabilities:** Log water intake without internet using Service Workers.
- **Push Notifications:** Automatic reminders based on user-set schedules.
- **Installable UI:** Desktop and mobile home screen integration (Manifest.json).
- **Data Visualization:** Daily and weekly hydration charts.
